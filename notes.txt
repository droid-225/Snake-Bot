Made a virtual environment:
python -m venv venv

to activate it in powershell:
./venv/Scripts/Activate.ps1

to activate in command prompt:
venv\Scripts\activate.bat

to deactivate or exit venv:
deactivate

installations required:
pip install pygame
pip3 install torch torchvision [just go to the pytorch website if you don't know what to do]
pip install matplotlib ipython
